# Hangout Server Ticket System

## Opening a Ticket
{: .text-red-300 }

Use `;ticket new Reason` to open a ticket. Replace "Reason" with the subject of your ticket.

## Closing a ticket (Staff Only)
{: .text-red-300 }

Staff have the ability to close tickets. To do so, go in the desired ticket and type `;ticket close`.
